## About this repository 
This repository represents the archive of user content from the popular plattforms _Steam_ and _Civ Fanatics_ analyzed in my master's thesis, written at the University of Regensburg (Germany). The folder _Civ Fanatics_ holds user content from [forums.civfanatics.com](https://forums.civfanatics.com/) and the folder _Steam_ from [steamcommunity.com](https://steamcommunity.com/). 

The user content cited in my thesis follows this example: [username] via [platform], [media] von [username] ([year of publication]).
